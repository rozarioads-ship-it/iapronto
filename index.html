Dúvidas

Perguntas frequentes

Preciso saber usar IA para usar o IApronto? +
Não. O IApronto foi feito exatamente para quem não sabe escrever prompts. Você responde perguntas simples ou descreve o que precisa em palavras normais — a ferramenta faz o resto.
O IApronto funciona com qual IA? +
Os prompts gerados são otimizados para o ChatGPT (versão gratuita e paga), mas funcionam muito bem no Gemini, Claude e qualquer outra IA de texto. A IA interna da ferramenta usa Claude para gerar e refinar prompts.
Como recebo o produto após a compra? +
Imediatamente após o pagamento aprovado, você recebe o acesso por e-mail pelo Kiwify. O produto é um arquivo HTML que você salva no celular ou computador — não precisa instalar nada.
Precisa de internet para usar? +
O gerador guiado e a biblioteca funcionam 100% offline. A IA integrada (IA Direta, Modo Chat, Score, variações) precisa de internet para conectar à API. Se estiver sem internet, ainda usa todas as funções locais.
Qual a diferença entre Básico e Pro? +
O Básico tem o gerador guiado e 7 prompts prontos — suficiente para começar. O Pro adiciona IA integrada (que gera prompts por você), Modo Chat, biblioteca com 26 prompts, análise de imagem, histórico, score de qualidade e variações. Se você vai usar todo dia, o Pro vale muito mais.
Posso usar para qualquer tipo de negócio? +
Sim. O IApronto tem nichos específicos para papelaria, moda, alimentação, beleza, saúde, educação, serviços, tecnologia, advocacia e contabilidade — mas funciona para qualquer negócio. Você também pode usar no modo genérico sem selecionar nicho.
